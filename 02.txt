Character user interface
Graphical User interface

하드웨어 위에 OS 위에 프로그램
OS는 커널과 시스템콜 인터페이스 두 층으로 나뉘어짐
프로그램이 OS에게 요청을 할 떄 요청하는 통로가 시스템콜인터페이스=시스템라이브러리
